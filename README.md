# MECHTRON 2TA4: Embedded Systems Design I
## Overview
The hardware used was the STM32F429-Discovery microcontroller. The MCU was programmed in C++ using the Arm Keil Studio software environment. The following 5 labs were performed:

- Lab 1: Introduction to Embedded Programming
- Lab 2: Reaction Time Tester
- Lab 3: Serial Communication and Non-volatile Memory
- Lab 4: Digital Thermometer and PWM Fan Controller
- Lab 5: Control of Stepper Motor

## Term Project
My embedded system project involved designing a two-way Morse code translator.
